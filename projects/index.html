Interactive Systems Project

I worked in a group of 5 engineers to construct a visual copy of the first level Super Mario Brothers. The game was created using C# in Visual Studio. We used the XNA framework as well. This project called for emphasis on code organization and maintainability as well as functionality. I learned how to write efficient and easily editable code. I also learned the basics of two-dimensional game design and techniques.

Various Arduino Projects

I purchased an Arduino Uno a few years ago because of their vast capabilities and creative flexibility. I have created a number of small projects for myself using the various components I have purchased. These range from small joystick-driven instruments to small LED light shows. I hope to get started on a larger, more permanent project soon.

Various Java Projects

I have worked on a number of personal projects using Java in the Eclipse IDE. I have created text based games and a couple simple AI projects to name a few. I am currently working on a Fantasy Football Calculator Project that should store players, stats, and trends and then be able to select an ideal and affordable team for popular fantasy football apps.

MichaelMatonis.com

I created this website in order to learn about web development. My goal was to learn a great deal about HTML, CSS, and JavaScript. I did not use a popular framework like Bootstrap because, while that would have been more efficient, I wanted to learn about these languages from the base up to gain a more full understanding. In addition to programming the site, I set up my own server using Apache 2.4, purchased a domain name, and learned a great deal about networking as well. I was happy to learn a lot about different web design techniques as well.
